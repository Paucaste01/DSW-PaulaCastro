Propósito del sitio

El propósito de este sitio es conocer la Sintaxis y selectores CSS, los cuales definen casi todo el aspecto visual de un sitio, allí podemos crear reglas de como queremos mostrar la información y crear un sitio agradable para los usuarios, logrando una grata experiencia en la Interacción Humano-Computadora.

Contenido:
Sintaxis: Una regla CSS siempre tiene un selector (el elemento al que quiero modificar) y una declaración (las características que le quiero agregar o cambiar). La declaración va encerrada entre llaves { } y dentro de ella se escriben todas las propiedades con sus valores, que modificarán al selector. Cuando hay más de una propiedad, se separan con punto y coma.
Selectores: El selector es la parte de la regla que asocia un elemento HTML con su diseño. Básicamente los selectores se pueden clasificar en:
  • Sector Etiqueta
  • Sector de ID
  • Sector de clase
  • Agrupación de Selectores
